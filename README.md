# 소상공인 대안 신용평가 모델

## 프로젝트 개요
소상공인의 부도 위험을 예측하는 대안 신용평가 모델입니다.
전통적인 CB 신용점수 대신 매출 패턴과 거래 데이터를 활용했습니다.

## 사용 기술
- Python, LightGBM, SHAP
- 피처 엔지니어링, 불균형 데이터 처리

## 주요 결과
- AUROC 0.97 달성
- 직접 설계한 피처(연체심각도점수, 부채x소진율)가 SHAP 중요도 1~2위
- 신용등급 10분위 분류 → 10등급 실제 부도율 59%

## 피처 엔지니어링
| 피처 | 설명 | 설계 의도 |
|------|------|-----------|
| 연체심각도점수 | 단기×1 + 중기×2 + 장기×4 | 연체 기간별 가중합 |
| 부채x소진율 | 부채비율 × 한도소진율 | 자금 압박 상호작용 |
| 연체이력존재 | 연체 있다/없다 이진화 | 상환 의지 신호 |
| 월매출_로그 | log(1 + 월매출) | 우측 꼬리 분포 정규화 |
